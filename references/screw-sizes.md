# 螺丝/螺母/热熔螺母规格参考

| 螺丝 | 紧配孔 | 标准孔 | 松配孔 | 沉头孔直径 | 沉头深度 |
|------|--------|--------|--------|-----------|---------|
| M1.6 | 1.7mm  | 1.8mm  | 2.0mm  | 3.0mm     | 1.0mm   |
| M2   | 2.1mm  | 2.2mm  | 2.4mm  | 3.8mm     | 1.2mm   |
| M2.5 | 2.6mm  | 2.7mm  | 2.9mm  | 4.7mm     | 1.5mm   |
| M3   | 3.1mm  | 3.2mm  | 3.4mm  | 5.6mm     | 1.7mm   |
| M4   | 4.1mm  | 4.3mm  | 4.5mm  | 7.5mm     | 2.3mm   |
| M5   | 5.1mm  | 5.3mm  | 5.5mm  | 9.2mm     | 2.8mm   |

| M8   | 8.1mm  | 8.4mm  | 8.6mm  | 14.5mm    | 4.4mm   |

**选择指南**:
- **紧配孔**: 螺丝需要自攻或紧密配合时使用
- **标准孔**: 3D 打印推荐使用，基于 ISO 273 精配合 (H12) 公差
- **松配孔**: 需要调整位置余量时使用

| M2.5 | 5.0mm   | 5.8mm   | 2.0mm   |
| M3   | 5.5mm   | 6.3mm   | 2.4mm   |
| M4   | 7.0mm   | 8.1mm   | 3.2mm   |
| M5   | 8.0mm   | 9.2mm   | 4.0mm   |
| M6   | 10.0mm  | 11.5mm  | 5.0mm   |
| M8   | 13.0mm  | 15.0mm  | 6.5mm   |

**螺母槽设计**（3D 打印用）:
```openscad
module nut_cavity(diameter, thickness, tolerance = 0.2) {
    // 六角槽 — 打印时水平方向有膨胀，对角方向需更多余量
    linear_extrude(thickness + 0.5)  // +0.5mm 确保螺母能嵌入
        circle(d = diameter + tolerance, $fn = 6);
}
```

## 热熔螺母 (Heat-set Inserts)

适用于 3D 打印件的螺纹嵌入。数据基于常见的黄铜热熔螺母。

| 螺丝 | 安装孔直径 | 安装深度 | 熔入温度 |
|------|-----------|---------|---------|
| M2   | 3.2mm     | 3.5mm   | ~210°C  |
| M2.5 | 3.8mm     | 4.0mm   | ~210°C  |
| M3   | 4.2mm     | 5.0mm   | ~210°C  |
| M4   | 5.5mm     | 6.0mm   | ~220°C  |
| M5   | 6.5mm     | 7.0mm   | ~230°C  |
| M6   | 7.5mm     | 8.0mm   | ~240°C  |

**热熔螺母安装孔设计**:
```openscad
module heatset_hole(outer_d, depth, taper_h = 1) {
    // 锥形导入口 + 直孔，便于螺母定位
    union() {
        cylinder(h = depth, d = outer_d);
        translate([0, 0, depth])
            cylinder(h = taper_h, d1 = outer_d, d2 = outer_d + 0.5);
    }
}
```
